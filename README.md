# Projeto: InfoTech IFPB - Soluções em Tecnologia

Projeto desenvolvido para a disciplina de **Desenvolvimento Web**, no 3º ano do curso Técnico Integrado em Informática do **IFPB**.
**Professor Orientador:** John Lenon

---

## 1. Sobre o Projeto

O **InfoTech IFPB** é uma plataforma web estática desenvolvida para simular a presença digital de uma empresa de suporte técnico de elite. O objetivo principal é fornecer uma interface intuitiva e profissional para que clientes possam conhecer os serviços oferecidos (Hardware, Software, Redes) e solicitar manutenção de forma rápida e eficiente.

A aplicação foca na experiência do usuário (UX), apresentando um design moderno, responsivo e semântico, alinhado com as melhores práticas de desenvolvimento web.

## 2. Requisitos do Sistema

### Requisitos Funcionais (RF)
*   **RF01 - Visualizar Serviços:** O usuário deve conseguir visualizar os serviços oferecidos com descrições claras e imagens ilustrativas.
*   **RF02 - Abrir Chamado:** O usuário deve conseguir preencher um formulário de solicitação de serviço com validação de dados.
*   **RF03 - Conhecer a Equipe:** O sistema deve apresentar a equipe técnica com fotos e descrições profissionais.
*   **RF04 - Navegação Intuitiva:** O usuário deve conseguir navegar facilmente entre as páginas Início, Serviços, Chamado e Equipe.

### Requisitos Não Funcionais (RNF)
*   **RNF01 - Responsividade:** O site deve se adaptar a diferentes tamanhos de tela (Desktop, Tablet, Mobile).
*   **RNF02 - Semântica HTML:** O código deve seguir os padrões semânticos do HTML5 para melhor acessibilidade e SEO.
*   **RNF03 - Performance:** O carregamento das páginas deve ser rápido, com imagens otimizadas.
*   **RNF04 - Usabilidade:** A interface deve ser limpa, com contraste adequado e tipografia legível.

## 3. Escolhas de Design

O design do **InfoTech IFPB** foi concebido para transmitir **confiança, profissionalismo e modernidade**.

*   **Paleta de Cores:**
    *   **Verde IFPB (`#007746`):** Cor primária, reforçando a identidade institucional e transmitindo estabilidade.
    *   **Laranja (`#e67e22`):** Cor de destaque (CTA), utilizada para chamar a atenção em botões e ações importantes.
    *   **Cinza Claro (`#f4f4f4`):** Fundo suave para garantir a legibilidade do conteúdo.
*   **Tipografia:** Uso de fontes sem serifa (System UI) para uma leitura limpa e moderna em telas digitais.
*   **Layout:**
    *   **Landing Page:** Estrutura focada em conversão, com Hero Section e ícones SVG para destacar benefícios.
    *   **Cards:** Utilizados nas páginas de Serviços e Equipe para organizar a informação de forma modular.

## 4. Tecnologias Utilizadas

*   **HTML5:** Estruturação semântica do conteúdo (`<header>`, `<main>`, `<article>`, `<footer>`).
*   **CSS3:** Estilização avançada com:
    *   **Flexbox:** Para alinhamentos unidimensionais (Header, Navbar).
    *   **CSS Grid:** Para layouts bidimensionais (Galeria de Serviços e Equipe).
    *   **CSS Variables:** Para consistência de cores e fácil manutenção.
    *   **Media Queries:** Para garantir a responsividade em dispositivos móveis.
*   **Git/GitHub:** Controle de versão e hospedagem.

## 5. Como Rodar Localmente

Este é um projeto estático, não requerendo instalação de dependências complexas.

1.  **Clone o repositório:**
    ```bash
    git clone https://github.com/Pedro-Arthur13/desenvolvimento-web.git
    ```
2.  **Navegue até a pasta do projeto:**
    ```bash
    cd desenvolvimento-web
    ```
3.  **Abra o projeto:**
    Basta abrir o arquivo `index.html` em qualquer navegador moderno (Chrome, Firefox, Edge).
